# QuickLink Portal: Barangay Ubaliw Digital Governance System

**QuickLink Portal** is a QR-based community information and service system designed for **Barangay Ubaliw, Polangui, Albay**. This project serves as a digital bridge between local government units and residents, streamlining document requests and community reporting.

## ‚ú® Key Features
* **Digital Resident Profile:** Secure login and registration for community members.
* **Document Request System:** Online application for Barangay Clearance and Certificates of Indigency.
* **Admin Command Center:** A dedicated dashboard for barangay officials to approve or reject requests with custom remarks.
* **Real-time Updates:** Powered by Firebase Firestore for live status tracking.
* **QR Integration:** Quick access to the portal via community-distributed QR codes.
* **Emergency Directory:** Instant access to local hotlines (PNP, BFP, Health Center).
